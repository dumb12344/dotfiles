# Arch Dotfiles
Intended to be installed after a minimal system is setup (e.g. archinstall with minimal profile)

```
sudo pacman -Sy git base-devel --needed
git clone https://github.com/dumb12344/dotfiles.git
cd dotfiles
make
```
